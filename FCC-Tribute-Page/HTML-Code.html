Microsoft Tribute Page

Bill Gates, pictured right, and Paul Allen, pictured left, the two founders of Microsoft and leaders of the personal computer revolution.

  • 1975: Microsoft was founded by Paul Allen and Bill Gates.
  • 1981: Through a contract with IBM Microsoft releases their first operating system, MS-DOS.
  • 1985: Microsoft devolops the Windows Operating System their GUI, or Graphical User Interface.
  • 1986: Microsoft stock goes public making four people billionaires and twelve thousand people millionaires
  • 1990: Microsoft introduces Microsoft Office, bringing many popular programs such as Word, Excel, and powerpoint to our computers.
  • 1994: Microsoft introduces Windows 95, with new additions such as, the famous start button, Internet Explorer, and Microsoft MSN.
  • 1996: Through a partnership with General Electric, Microsoft expands into the cable television market by creating the 24/7 news channel MSNBC.
  • 2000: Bill Gates steps down as CEO of Microsoft for the new position of Chief Software Architect, giving the CEO position to Steve Ballmer.
  • 2001: Microsoft releases both Windows XP and a brand new gaming console, the Xbox.
  • 2007: Microsoft releases Windows Vista, which was critized over the high cost, lacking performance, and new security features.
  • 2008: Bill Gates retires from his role as Cheif Software Architect.
  • 2009: The first Microsoft Store opened in Scottsdale Arizona, providing customers an in-person experiance with Microsoft products.
  • 2010: Microsoft introduces the Windows Phone OS and a new design languange focused on simplicity
  • 2012: Microsoft introduces the Surface Tablet, which is the first computer to have it's hardware made by Microsoft.
  • 2014: Steve Ballmer steps down as CEO of Microsoft, handing the position to Satya Nadella and John Thomson took Bill Gate's role as chairman.
  • 2016: Microsoft announces the merger of the personal computer and Xbox divisions in favor of a Universal Windows Platform.
  • 2018: Microsoft announces the acquisition of Github, which is expected to finalize at the end 2018
